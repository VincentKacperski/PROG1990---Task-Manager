Details about the Task Manager application

- The porgram uses void func to perform all tasks
- A void func is used to display the main menu
- The programs uses a switch to perform changes

Overview: The program performs the following tasks...

- Display all manager tasks
- Add a new task at the beginning
- Add a new task at the end
- Add a new task at a point
- Delete a task at the beginning
- Delete a task at the end
- Delete a task at a point
- Exit the program
